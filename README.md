# Java Mastery Roadmap Tracker

A dynamic, interactive roadmap for mastering Java, powered by Firebase Firestore.

## Features
-   **Dynamic Content**: Roadmap steps and phases are fetched from Firestore.
-   **Progress Tracking**: Your progress is saved automatically to the cloud.
-   **Real-time Stats**: View your completion percentage and velocity.

## Setup
1.  Clone this repository.
2.  Open `index.html` in your browser.

## Customization
To use your own database, update the `firebaseConfig` in `script.js`.
